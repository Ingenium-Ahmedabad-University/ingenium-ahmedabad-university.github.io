Ingenium Kaggle Competition

If you are a Data Science and Machine Learning enthusiast, get ready for an amazing competition we have for you. Kaggle allows users to find and publish data sets, explore and build models in a web-based data-science environment, work with other data scientists and machine learning engineers, and enter competitions to solve data science challenges. We will be conducting the competition on Kaggle platform. So don’t miss the opportunity as we have some exciting prizes for the winners!

Prize: Worth 10K

Rules:

1. Every participant must have a ​Kaggle​ account.
2. Maximum allowed team size is ​3
3. The competition will last for 4 days.
4. Each team will be able to make 8 submissions per day.
5. There will be a public leaderboard that will be available during the competition and a private leaderboard that will be the final standings. During the competition, each team will be put onto the public leaderboard and the private leaderboard will be made available only at the end of the competition.
4. ​Each team will have to select one solution that they consider final which will be evaluated for the final standings.
5. ​Use of any publicly available library / package is allowed. Use of pretrained models is not permitted. Use of external datasets is not allowed.

Certificate Policy:

1. Every participant must have a ​Kaggle​ account.
2. Maximum allowed team size is ​3
3. The competition will last for 2 weeks
